# Sum Array

Given an array of integers, print the sum of its elements.

## Input

The first line contains an integer `n`.
The second line contains `n` integers.

## Output

Print one integer: the sum of the array.

## Constraints

- `1 <= n <= 200000`
- `-10^9 <= a_i <= 10^9`

## Sample

```text
5
1 2 3 4 5
```

```text
15
```
